### Installation

1. Git Clone 
    
    ```bash
    $ git clone https://github.com/cau-bootcamp/wehavit.git 
    $ cd wehavit
    ```

2. Install the dependencies

    ```bash
    flutter pub get
    ```

3. Run wehavit

    ```bash
    flutter run
    ```
